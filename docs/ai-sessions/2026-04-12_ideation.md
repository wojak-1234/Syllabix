# AI 협업 세션: 기획 및 아키텍처 (2026-04-12)

## 세션 요약
- **참여자**: 사용자(개발자), Antigravity(AI)
- **목표**: MVP 범위 확정, 기술 스택 최종화, AI 협업 규칙 수립.
- **주요 성과**: 범용 교육 플랫폼에서 '소크라테스식 AI 페어 프로그래머'를 갖춘 코딩 특화 교육 플랫폼으로 방향을 구체화함.

## 주요 논의 사항

### 1. 프로젝트 명명 및 정체성
- **결정**: CodeMentor AI (코드멘토 AI)
- **정체성**: "답을 주는 AI가 아닌, 깨달음으로 이끄는 AI."

### 2. 기술 스택 확정
- **프런트엔드**: Next.js 14 (App Router), Monaco Editor
- **백엔드/AI**: Gemini API, LangChain
- **데이터베이스**: Supabase (PostgreSQL + pgvector)
- **결정 근거**: 높은 처리 속도와 강력한 추론 능력을 가진 Gemini를 채택하고, 에이전트 기능을 위해 LangChain 기반의 확장이 용이한 구조를 선택함.

### 3. API 라우트 구조 설계
- 학습 루프를 반영하여 다음과 같이 API 경로를 단일화함:
  - `onboarding`: 초기 수준 진단
  - `curriculum/generate`: 맞춤형 로드맵 생성
  - `chat`: 소크라테스식 튜터링 대화
  - `analytics/blind-point`: 선제적 취약점 탐지
  - `lectures`: RAG 기반 강의 콘텐츠 검색
