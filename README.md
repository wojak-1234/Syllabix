# 🚀 Syllabix

AI 기반 차세대 교육 솔루션으로,  
교사와 학습자를 위한 **맞춤형 커리큘럼 생성 및 학습 최적화 플랫폼**입니다.

---

## 📌 프로젝트 개요

Syllabix는 교육 현장에서 발생하는 문제를 해결하기 위해  
AI를 활용하여 다음을 제공합니다:

- 개인 맞춤형 커리큘럼 자동 생성
- 학습 데이터 기반 분석 및 피드백
- 교사 중심의 효율적인 수업 설계 지원

---

## 🎯 문제 정의

기존 교육 시스템의 한계:
- 획일적인 커리큘럼
- 학습자 수준 반영 부족
- 교사의 높은 수업 설계 부담

👉 Syllabix는 이를 해결합니다.

---

## 💡 주요 기능

### 1. AI 커리큘럼 생성
- 학습자의 수준 및 목표 기반 자동 커리큘럼 설계

### 2. 학습 분석
- 학습 데이터 기반 성취도 분석
- 취약 개념 자동 탐지

### 3. 실습 & 문제 최적 배치
- 학습 효율을 고려한 문제 수 및 실습 구성

### 4. 교사 페르소나 기반 설계
- 교사의 스타일을 반영한 커리큘럼 생성
